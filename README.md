# Symmetric Attention Decomposition — Hopfield Perspective

> **Balancing Fidelity and Diversity in Diffusion Models via Symmetric Attention Decomposition: Hopfield Perspective**
>
> [Hyunmin Cho](https://github.com/hyeon-cho)<sup>1</sup>, Woo Kyoung Han<sup>1</sup>, [Kyong Hwan Jin](https://ipa.korea.ac.kr/)<sup>1†</sup>
>
> <sup>1</sup>Department of Electrical Engineering, Korea University, Seoul, South Korea
>
> <sup>†</sup>Corresponding author
>
> **ICML 2026**

[![Code](https://img.shields.io/badge/Code-GitHub-181717.svg?logo=github)](https://github.com/hyeon-cho/Attention-Symmetric-Decomposition)

This repository hosts the **project page** for our ICML 2026 paper. We characterize the pre-softmax attention matrix $\mathbf{QK}^\top$ in transformers as an associative memory matrix, decompose it into symmetric (energy landscape) and skew-symmetric (circulation) parts, and use the skew component as a training-free knob that breaks metastable mixtures in diffusion sampling.
